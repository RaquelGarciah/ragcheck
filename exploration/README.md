# exploration/

Borrador: aquí se prueban ideas, ablaciones y experimentos sueltos (notebooks
incluidos). **Nada de esta carpeta es citable en la memoria** ni tiene que pasar
los tests ni el mismo estándar de calidad que `src/ragcheck/` y `scripts/`.

Cuando una idea probada aquí se consolida, su versión limpia pasa a la librería
(`src/ragcheck/`) o a un script atómico (`scripts/`), y esa es la que se cita.
